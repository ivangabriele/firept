<h1 align="center">FirePT</h1>
<h3 align="center">A ChatGPT plugin server to give ChatGPT extra powers.</h3>

---

## Get started

### Install

```sh
npm i -g firept
```

### Run

```sh
firept
```

## Features

### Web Features

- [x] Searching (Brave Search)
- [x] Fetching web pages

### Current Working Directory Features

- [ ] Listing files
- [ ] Reading files
- [ ] Writing files
- [ ] Running bash scripts
